Extends WooCommerce with B2B/B2C functionalities and integrates with Zoho, complementing the WooCommerce Zoho Integration plugin.

== Description ==

WooCommerce Zoho B2B Manager provides a comprehensive suite of tools for businesses looking to manage B2B and B2C sales through WooCommerce, with seamless integration with the Zoho ecosystem. This plugin is designed to work alongside the "WooCommerce Zoho Integration" plugin, enhancing its capabilities with specific B2B features.

**Key Features (Planned):**

*   B2B User Registration and Application System
*   Customizable B2B User Roles (e.g., Wholesaler, Distributor)
*   Tiered/Role-Based Pricing and Discounts
*   Product Visibility Control (B2B/B2C, Guest)
*   Minimum Order Quantities
*   Quick Order Forms / Wishlist Functionality
*   Integration with Zoho CRM/Books for B2B data (Contacts, Companies, Sales Orders)
*   Admin panel for managing B2B settings, applications, and Zoho sync.

== Installation ==

1.  Upload the `woocommerce-zoho-b2b-manager` folder to the `/wp-content/plugins/` directory.
2.  Activate the plugin through the 'Plugins' menu in WordPress.
3.  Ensure WooCommerce is installed and activated (version 9.0+ recommended).
4.  (Recommended) Ensure the "WooCommerce Zoho Integration" plugin (github.com/axelbriones/woocommerce-zoho-integration) is installed and configured if you wish to use its Zoho connection capabilities.
5.  Navigate to WooCommerce > Zoho B2B Settings (or similar, actual menu item TBD) to configure the plugin.

== Frequently Asked Questions ==

*   **Does this replace the existing WooCommerce Zoho Integration plugin?**
    No, this plugin is designed to complement and extend the functionalities of the existing `woocommerce-zoho-integration` plugin, focusing on B2B specific features.

*   **What versions of WooCommerce and WordPress are required?**
    Requires WordPress 6.0+ and WooCommerce 9.0+. PHP 7.4+ is also required.
